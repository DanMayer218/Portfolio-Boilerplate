PRISM

Visualizing Large Sets of Geo-Spatial Data

PRISM is a visual tool that provides new perspectives on trends in property sales in New York City's five boroughs. PRISM harnesses Kepler GL to leverage over 200,000 property sales provided by NYC open data. Prism can accept diverse datasets that follow geo-spatial formats.

BOOtIQUE

A peer reviewed portfolio forum for Bootcamp students

BOOTIQUE is a MERN app that provides Coding Bootcamp students with a forum to exhibit their professional materials and work from class. The forum provides a creative environment for other students to post constructive criticism and share insights.

Node Your Code

Test your coding fundamentals

A coding Bootcamp project written in Javascript and meant to test your coding fundamentals.

VACAY

Everything you need to know, Everywhere you want to go.

VACAY provides the professional traveller with an essential set of informative tools. Built upon a series of AJAX calls VACAY provides the user with local information, weather reports, currency exchange and maps for the travellers destination. Additionally there is an itiniery maker for the user to keep track of important travel information.
